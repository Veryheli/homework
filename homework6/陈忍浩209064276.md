#### 5（8）编写算法交换二叉树中所有节点的左右子树。
```c++
template<class T>
void BiTree<T>::shiftChilds(node<T> *bt)
{
    if(bt == NULL) return;
    node<T> * temp = bt->lchild;
    bt->lchild = bt->rchild;
    bt->rchild = temp;
    shiftChilds(bt->lchild);
    shiftChilds(bt->rchild);
}
```
